# Обновление иконок и UI

## Что было:
- Использовался стандартный набор иконок `ic_launcher` (PNG).
- Отсутствовали современные Adaptive Icons в виде векторов.
- Отсутствовала монохромная иконка (Themed icon) для Android 13+.

## Что сделано:

1. **Созданы новые адаптивные иконки** (Adaptive Icons) на базе существующей компасной сплэш-иконки:
   - `ic_launcher_background.xml` (сплошной фон `#1A1C16`)
   - `ic_launcher_foreground.xml` (вписанный векторный компас)
   - `ic_launcher_monochrome.xml` (поддержка тематических иконок Android 13+)

2. **Обновлена структура ресурсов**:
   - Обновлены файлы `ic_launcher.xml` и `ic_launcher_round.xml` в папке `mipmap-anydpi-v26`.

3. **Сгенерированы исходники**:
   - `app/src/main/assets/icon_source.svg` (мастер-копия иконки)
   - `app/src/main/assets/playstore-icon.svg` (1024x1024 SVG для Play Store)

4. **Добавлены инструменты**:
   - `tools/export_icons.ps1` — скрипт автоматической конвертации SVG в набор PNG для старых устройств через Inkscape.
   - `tools/README_ICON.md` — инструкции по работе с инструментами.

## Визуальная оценка UI (на реальном устройстве)
Проведено тестирование на устройстве:
Оригинальное приложение (`com.example.teamcompass`) было собрано (`installDebug`), запущено, и сделаны скриншоты (`screenshots/home.png`, `screenshots/menu.png`).

**Оценка:**
- Проект использует кастомную **DayNight** тему на основе `MaterialComponents`.
- Цветовая палитра выдержана в качественных, приглушённых "multicam" оттенках, которые хорошо сочетаются между собой (зелёный `#5F6B4A`, текстовый `#E8E4D5`, фон `#1A1C16`).
- Очевидно применение кастомного **Splash Screen API (Android 12+)**, что является отличной практикой (анимация 350мс с кастомной иконкой `ic_compass`).
- Отключённые системные бары (transparent status/navigation bar) предполагают edge-to-edge дизайн, что отлично смотрится на полноэкранных интерфейсах.
- **Рекомендации:** Убедитесь, что контраст текста (особенно `tc_text_secondary`) достаточен на всех вариантах `surface`, так как в полевых условиях читаемость критически важна. Композиция Adaptive Icons хорошо гармонирует с общей темой.
